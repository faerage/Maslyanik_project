Завдання № 1

Написати програму, яка обчислює значення функції Гауса: \[f(x) = \frac{1}{\sigma \sqrt{2 \pi}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}\]

Code 
import math

def gaussian_function(x, mu, sigma):
    coefficient = 1 / (sigma * math.sqrt(2 * math.pi))
    exponent = -((x - mu) ** 2) / (2 * sigma ** 2)
    result = coefficient * math.exp(exponent)
    return result

mu = float(input("Введіть значення mu: "))
sigma = float(input("Введіть значення sigma: "))

x = float(input("Введіть значення x: "))

result = gaussian_function(x, mu, sigma)

# Виведення результату
print(f"Значення функції Гауса при x = {x}, mu = {mu}, sigma = {sigma}: {result}")
Введіть значення mu:  5
Введіть значення sigma:  12
Введіть значення x:  6
Значення функції Гауса при x = 6.0, mu = 5.0, sigma = 12.0: 0.0331299555215285

Ваше завдання – завершити код, щоб оцінити такий вираз: \[\frac{1}{x + \frac{1}{x + \frac{1}{x + \frac{1}{x + \frac{1}{x}}}}}\]

Результат має бути присвоєний y.

Ви можете використовувати додаткові змінні для зменшення виразу (але це не обов’язково). Ретельно тестуйте свій код.

Code 
x = float(input("Enter value for x: "))
n = 4
def calculate_expression(x, n):
    if n == 0:
        return 0
    else:
        return 1 / (x + calculate_expression(x, n - 1))


result = 1 / (x + calculate_expression(x, n))
print(result)
